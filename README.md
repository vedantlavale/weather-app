# Weather App
#### 📝 Description
Weather app that uses the VisualCrossing API. A JavaScript project made for the fun.

#### 💡 Features
* Search cities will appear as cards
*  Error popup if searched city could not be found
*  Mobile responsive

#### 🛠️ Built with
*  HTML
*  Tailwind CSS
*  JavaScript
*  VisualCrossing API
